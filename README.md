# Food Recipe Card - Menu Book 🍔🍕

A simple, interactive digital recipe book and menu card created as part of the Modern Web Application Development (MWAD) assignment. This webpage showcases various food items with quick-jump links to view detailed ingredients and step-by-step cooking preparation instructions.

## 🔗 Live Repository Link
* **GitHub Repository:** [Assignments-03--FizaBaneen-MWAD](https://github.com)

---

## 🚀 Features
* **Visual Gallery:** Displays high-quality images of popular local and international fast foods/dishes (Zinger Burger, Pizza, Biryani, Nihari, etc.).
* **Anchor Navigation:** Click on specific food display thumbnails to skip directly down to that specific recipe card on the page.
* **Custom Typography:** Integrated local standalone web fonts directly from the assets directory for stylized headings and interfaces.
* **Pure HTML/CSS:** Built fundamentally using core structuring concepts and local embedded font-face parameters.
